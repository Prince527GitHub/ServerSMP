To get a look of the commands in discord, use +help to view the commands!

Command Name Usage Description Aliases
help [command] Show the command list. h, command, commands, cmd, cmds
about no usage Send the information about the bot. botinfo, info, stats
invite no usage Send the bot's invite link. no aliases
nowplaying no usage Send information about the current music player. np, now-playing
pause no usage Pause the music player. no aliases
ping no usage Show the current ping of the bot. pong, pang, pung, peng, pingpong
play [youtube video or playlist link | youtube video name] Play some music. p, add, play-music
queue no usage Show the current queue. q
repeat [all | one | disable] Repeat current music or the queue. loop, music-repeat, music-loop
resume no usage Resume the music player. no aliases
skip no usage Skip the current music. s
stop no usage Stop the music player. st, disconnect, dc
volume [new volume] Show or change the music player's volume. vol
